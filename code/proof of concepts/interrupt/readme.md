# start/stop interrupt proof of concept
minimale hard- en software die de correcte werking van een start/stop drukknop aantoont, gebruik makend van een hardware interrupt

Sluit de drukknop en de led aan zoals op het elektrisch schema en laad de code in je arduino nano. Haal de kabel uit de arduino en zet de schakelaar dicht van de voeding. Wanneer je op de drukknop drukt zal de lamp aan gaan. Als er daarna nog eens op gedrukt wordt zal het uit gaan.
