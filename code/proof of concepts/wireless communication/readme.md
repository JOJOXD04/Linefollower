# draadloze communicatie proof of concept
minimale hard- en software waarmee aangetoond wordt dat duplex kan gecommuniceerd worden tussen de microcontroller en een smartphone, gebruik makend van [Serial Bluetooth Terminal]

### gebruiksaanwijzing
Sluit de bluetooth module en de led aan zoals op het elektrisch schema en laad de code in je arduino nano. Open de Serial Bluetooth Terminal op je smartphone. En verbind met de bluetooth module.
